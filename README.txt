Instalación y uso
Si quieres descargar y ejecutar el blog en local:

Clona el repositorio:
git clone https://tu-repo-url.git

Instala las dependencias (si aplica):
npm install o composer install

Ejecuta el servidor local:
npm start o el comando correspondiente a tu stack

Abre tu navegador en http://localhost:3000 (o el puerto que uses)
